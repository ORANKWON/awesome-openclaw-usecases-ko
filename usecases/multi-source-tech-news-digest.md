# 멀티소스 테크 뉴스 다이제스트

## 기능 설명
100개 이상의 소스에서 기술 뉴스를 자동 수집·랭킹합니다:
- RSS 피드 46개 (주요 기술 매체)
- Twitter/X 계정 44개 (핵심 인플루언서)
- GitHub 저장소 19개 (소프트웨어 릴리스)
- Brave API 웹 검색 4개

기사는 중복 제거 후 소스 우선순위, 교차 소스 언급 횟수, 최신성, 인게이지먼트로 품질 점수화됩니다. Discord, 이메일, 텔레그램으로 전달.

**선택적 API 토큰:** Twitter/X Bearer Token, Brave Search 키, GitHub 토큰

## 필요한 스킬
- Tech news digest 스킬 (ClawHub)
