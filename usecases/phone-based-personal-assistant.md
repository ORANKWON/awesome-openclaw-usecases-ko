# 전화 기반 개인 어시스턴트

## 문제점
스마트폰 앱이나 브라우저 없이 어떤 전화기에서든 AI 에이전트에 접근하고 싶습니다. 운전 중이나 걷는 중에 핸즈프리로 사용하고 싶습니다.

## 기능 설명
ClawdTalk으로 OpenClaw가 전화를 받고 걸 수 있습니다. 음성으로 캘린더 조회, Jira 업데이트, 웹 검색 지원. 전화 연결에 Telnyx 사용. SMS 지원 추가 예정.

## 프롬프트 예시

```
전화로 사용 가능합니다. 전화가 오면 인사하고 어떻게 도와드릴지 물어보세요.

캘린더 조회: "오늘 일정이 뭐야?"
Jira 업데이트: "내 열린 티켓 보여줘"
웹 검색: "AI 에이전트 최신 뉴스 검색해줘"
```

## 필요한 스킬
- [ClawdTalk](https://github.com/team-telnyx/clawdtalk-client)
- Calendar 스킬
- Jira 스킬
- 웹 검색
