# 세컨드 브레인

## 문제점
모든 노트 앱은 결국 부담이 됩니다. 정리하는 마찰이 잊어버리는 마찰보다 큽니다.

## 기능 설명
문자 메시지로 기억 저장, 커스텀 검색 가능한 Next.js UI로 확인:
- 텔레그램/iMessage/Discord로 무엇이든 전송 — 즉시 기억
- OpenClaw의 내장 메모리에 영구 저장
- 커스텀 Next.js 대시보드: 모든 기억/대화 목록, 전역 검색 (Cmd+K), 날짜·유형별 필터, 깔끔하고 미니멀한 UI
- OpenClaw가 전체 Next.js 앱을 직접 빌드하고 배포

## 설정 프롬프트

```
모든 메모, 대화, 기억을 검토할 수 있는 세컨드 브레인 시스템을 만들어 주세요.
Next.js로 구현해 주세요.

포함 내용:
- 모든 기억과 대화의 검색 가능한 목록
- 전체 검색 (Cmd+K)
- 날짜 및 유형별 필터
- 깔끔하고 미니멀한 UI
```

## 필요한 스킬
- Telegram/iMessage/Discord
- Next.js (OpenClaw가 직접 빌드)
