# 실적 발표 트래커

## 문제점
수십 개 기술 기업의 실적 시즌을 팔로우하려면 여러 소스를 확인하고 발표 날짜를 기억해야 합니다.

## 기능 설명
- 매주 일요일 미리 보기: 테크/AI 기업의 다음 주 실적 발표 일정 스캔, 텔레그램 "earnings" 토픽에 게시
- 사용자가 추적할 기업 선택 → OpenClaw가 각 실적 날짜에 단발성 크론 작업 예약
- 각 발표 후: 결과 검색, 요약 포맷 (예상치 달성/미달, 매출, EPS, 핵심 지표, AI 하이라이트, 가이던스)

## 설정 프롬프트

```
매주 일요일 오후 6시에 크론 작업 실행:
1. 다음 주 테크 및 AI 기업 실적 발표 일정 검색
2. 내가 관심 있는 기업 필터링 (NVDA, MSFT, GOOGL, META, AMZN, TSLA, AMD 등)
3. 텔레그램 "earnings" 토픽에 목록 게시
4. 추적할 기업 확인 대기

내가 추적할 기업을 답하면:
1. 각 실적 날짜/시간에 단발성 크론 작업 예약
2. 각 발표 후 결과 검색
3. 요약 포맷: 예상치 달성/미달, 매출, EPS, 핵심 지표, AI 하이라이트, 가이던스
4. 텔레그램 "earnings" 토픽에 게시

내가 주로 추적하는 기업을 기억해서 자동으로 제안해 주세요.
```

## 필요한 스킬
- `web_search` (기본 제공)
- 크론 작업
- Telegram 토픽
