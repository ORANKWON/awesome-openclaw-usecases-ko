# 이벤트 참석 확인 자동화

## 문제점
게스트에게 전화를 걸어 참석 여부를 확인하는 것은 반복적이고 시간이 많이 걸립니다.

## 기능 설명
SuperCall 플러그인을 사용한 자동 전화. AI가 이벤트 코디네이터("Jamie")로 자기 소개를 하고 참석 여부와 특별 요청을 수집합니다. 보안 측면: 음성 에이전트는 "제공된 컨텍스트에만 접근"하므로 광범위한 에이전트 시스템 접근을 차단합니다.

**팁:** 소규모 테스트 그룹으로 시작하고, 전화 시간대에 주의하며, Twilio 분당 요금을 모니터링하세요.

## 필요한 스킬
- SuperCall 플러그인
- Twilio 계정
- OpenAI API 키 (GPT-4o Realtime 음성용)
