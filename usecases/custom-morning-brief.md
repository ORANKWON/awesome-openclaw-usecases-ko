# 맞춤형 모닝 브리프

## 문제점
가장 생산적인 아침 시간을 단순히 현황 파악에 써버리는 동안 AI 에이전트는 밤새 대기 상태로 있습니다.

## 기능 설명
매일 설정한 시간(예: 오전 8시)에 텔레그램/Discord/iMessage로 완전히 맞춤화된 모닝 브리핑 전송. 포함 내용: 관심 뉴스, 당일 할 일(Todoist/Reminders에서), 전체 콘텐츠 초안(아이디어만이 아님), 선제적 태스크 추천.

## 설정 프롬프트

```
정기적인 모닝 브리프를 설정하고 싶습니다. 매일 오전 8시에
텔레그램으로 리포트를 보내주세요.

포함 내용:
1. 내 관심사 관련 뉴스 (AI, 스타트업, 테크)
2. 오늘 만들 수 있는 콘텐츠 아이디어
3. 오늘 완료해야 할 태스크 (to-do 리스트에서 가져오기)
4. 당신이 대신 처리할 수 있는 태스크 추천

콘텐츠 아이디어는 제목만이 아닌 전체 초안 스크립트/개요로 작성해 주세요.
```

**개인화:** 봇에 문자를 보내 포함 내용을 언제든 업데이트 가능.

## 필요한 스킬
- Telegram/Discord/iMessage
- Todoist/Apple Reminders/Asana
- `x-research-v2` (선택)
