# 자율 미니앱 빌더

## 기능 설명
목표 중심의 자율 태스크 생성. 개인·업무 목표를 "brain dump" 형태로 입력하면 에이전트가 목표 달성을 위한 일일 태스크 4–5개를 생성하고 실행합니다. 커스텀 Kanban 보드(To Do / In Progress / Done)로 진행 상황 확인. 목표에 따라 경량 앱을 밤새 빌드 가능.

**핵심 구현 인사이트:** 메인 세션과 서브에이전트가 같은 태스크 파일을 동시에 업데이트할 때 레이스 컨디션 발생. 해결책: 파일 분리 — `AUTONOMOUS.md`는 목표용(메인 세션 전용), `memory/tasks-log.md`는 서브에이전트용 추가 전용 로그.

## 필요한 스킬
- `sessions_spawn` / `sessions_send`
- 파일 시스템
