# 받은편지함 정리

## 문제점
뉴스레터가 읽히지 않은 채로 쌓입니다.

## 기능 설명
매일 저녁 8시에 지난 24시간 동안 받은 뉴스레터 이메일 다이제스트 전달. 피드백을 통해 취향을 학습하고, 향후 더 나은 다이제스트를 위해 메모리 업데이트.

## 설정 방법

1. 선택: OpenClaw 전용 Gmail 생성
2. Gmail OAuth 스킬 설치
3. 프롬프트:

```
매일 오후 8시에 크론 작업을 실행하여 지난 24시간 동안 받은
모든 뉴스레터 이메일을 읽고 가장 중요한 내용과 더 읽을 수 있는 링크와
함께 다이제스트를 보내 주세요. 그런 다음 내가 좋은 내용을 선별했는지
피드백을 요청하고, 내 선호도에 따라 메모리를 업데이트하세요.
```

## 필요한 스킬
- Gmail OAuth (clawhub.ai)
