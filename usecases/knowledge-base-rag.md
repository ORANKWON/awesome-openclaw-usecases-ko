# 지식 베이스 RAG

## 문제점
하루 종일 기사를 읽고 트윗을 보고 영상을 시청하지만 지난주에 봤던 그 내용을 다시 찾을 수가 없습니다.

## 기능 설명
- 텔레그램/Slack에 URL 드롭 → 콘텐츠 자동 수집 (기사, 트윗, YouTube 트랜스크립트, PDF)
- 시맨틱 검색: "에이전트 메모리에 대해 저장한 게 뭐야?" → 소스와 함께 관련 결과 반환
- 다른 워크플로우에 연결 (예: 영상 아이디어 파이프라인)

## 설정 프롬프트

```
"knowledge-base" 토픽에 URL을 드롭하면:
1. 콘텐츠 수집
2. 메타데이터와 함께 지식베이스에 저장 (제목, URL, 날짜, 유형)
3. 수집 확인 응답: 무엇이 수집됐는지, 청크 수

질문을 하면:
1. 지식베이스를 시맨틱으로 검색
2. 소스와 관련 발췌문과 함께 상위 결과 반환
3. 좋은 결과가 없으면 알려주기
```

## 필요한 스킬
- `knowledge-base` 스킬 (clawhub.ai, 또는 임베딩으로 커스텀 RAG)
- `web_fetch` (기본 제공)
- Telegram/Slack
