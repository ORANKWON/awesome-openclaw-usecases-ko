# 전화 알림

에이전트가 이미 당신을 위해 모니터링하고 있습니다 — 주식, 이메일, 스마트 홈, 캘린더 — 하지만 알림은 무시하기 쉽습니다. 푸시 알림은 쌓입니다. 채팅 메시지는 묻힙니다. 정말 중요한 것에는 스와이프할 수 없는 무언가가 필요합니다.

이 활용 사례는 에이전트에게 알림 채널로 전화를 제공합니다. 충분히 긴급한 것이 있으면 에이전트가 실제 전화번호로 전화를 걸어 무슨 일이 일어나고 있는지 알려주고, 당신은 되물을 수 있습니다. 양방향 대화, 로보콜이 아닙니다.

## 기능 설명

- 에이전트가 당신의 주의가 필요하다고 결정합니다 (가격 알림, 긴급 이메일, 약속 알림, 무엇이든)
- 에이전트가 [clawr.ing](https://clawr.ing)을 통해 전화를 걸며, 관리형 전화 서비스입니다 — Twilio 설정 없음, 구성할 API 키 없음
- 전화를 받으면 알림을 듣고 실시간으로 후속 질문을 할 수 있습니다
- 대화가 끝나면 통화가 종료됩니다

핵심 아이디어: 에이전트가 당신에게 전화합니다. 당신이 에이전트에게 전화하는 것이 아닙니다. 이것은 하트비트 체크, cron 작업 또는 모든 트리거와 작동합니다 — 에이전트가 전화할 가치가 있는지 평가하고 행동합니다.

## 왜 이것이 작동하는가

OpenClaw 에이전트는 이미 주도권을 가지고 있습니다 — 하트비트, cron, 이벤트 트리거. 하지만 출력 채널은 확인하지 않을 수 있는 채팅 플랫폼으로 제한됩니다. 전화는 특히 책상에서 멀리 떨어져 있을 때 확실히 주의를 끄는 하나의 알림 채널입니다.

clawr.ing은 전화 인프라를 처리합니다. 설정 프롬프트를 하나 붙여넣으면 에이전트가 전화를 걸 수 있는 능력을 얻습니다. Twilio 계정 없음, 전화번호 프로비저닝 없음, 웹훅 구성 없음. 서비스는 실제 PSTN 통화(VoIP 오버레이가 아님)로 100개 이상의 국가를 커버합니다.

## 필요한 스킬

- [clawr.ing](https://clawhub.ai/marcospgp/clawring) — [clawr.ing 대시보드](https://clawr.ing)에서 설정 프롬프트를 OpenClaw 채팅에 붙여넣어 설치합니다. CLI 설치 불필요.

그게 다입니다. 다른 의존성 없음. 설정 프롬프트에는 API 키와 스킬 문서 링크가 포함되어 있습니다 — 에이전트가 읽고 나머지를 파악합니다.

## 예시: 모닝 브리핑 전화

매일 아침 개인화된 브리핑으로 전화하는 cron 작업을 설정하세요:

```
매주 평일 오전 7:30에 모닝 브리핑으로 전화해줘. 포함할 것:
- 내 도시의 날씨 예보
- 오늘의 내 캘린더
- 밤새 들어온 긴급 이메일
- 내 관심사와 관련된 주요 3개 뉴스 헤드라인

간결하게 유지 — 2분 이내를 목표로 해. 질문하면 답해줘.
받지 않으면 재시도하지 마.
```

## 예시: 가격 알림

에이전트에게 무엇을 감시하고 언제 전화할지 알려주세요:

```
NVDA 주가를 모니터해줘. 하루에 5% 이상 떨어지면
즉시 전화해서 무슨 일이 일어났는지 알려줘. 하락을 설명할 수 있는
관련 뉴스를 포함해줘.
```

## 예시: 긴급 이메일 필터

```
업무 시간 동안 15분마다 내 받은편지함을 확인해줘.
상사의 이메일이나 긴급 표시된 이메일을 보면
요약해서 전화해줘. 다른 모든 것은 그냥 채팅 메시지를 보내.
```

## 핵심 인사이트

- **과도하게 사용하지 마세요.** 요점은 전화가 "이것이 실제로 중요하다"는 것을 의미한다는 것입니다. 에이전트가 하루에 10번 전화하면 무시하기 시작할 것입니다. 전화할 만한 것과 채팅할 만한 것의 명확한 임계값을 설정하세요.
- **하트비트나 cron과 짝을 이루세요.** clawr.ing은 전달 채널입니다 — 여전히 트리거가 필요합니다. 하트비트 체크(30분마다)는 모니터링 작업에 작동합니다. Cron 작업은 예약된 브리핑에 작동합니다.
- **양방향 대화.** 푸시 알림과 달리 통화에서 후속 질문을 할 수 있습니다. "잠깐, 어떤 이메일?" 또는 "지금 현재 가격은?" — 에이전트가 실시간으로 응답합니다.
- **통화에는 빠른 모델.** 전화 대화는 빠른 응답이 필요합니다. OpenClaw가 스킬 수준 모델 라우팅을 지원하면 clawr.ing을 빠른 모델(Haiku급)에 할당하세요. 생각하는 소리가 지연을 커버하지만 빠를수록 항상 더 좋습니다.
- **프라이버시.** clawr.ing은 녹음이나 전사를 저장하지 않습니다. 오디오는 전송 중 암호화되고 처리 후 폐기됩니다.
