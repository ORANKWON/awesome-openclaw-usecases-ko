# 시맨틱 메모리 검색

## 문제점
OpenClaw 마크다운 메모리 파일이 쌓일수록 특정 과거 결정을 찾기가 번거로워집니다. 키워드 검색은 의미적으로 관련된 콘텐츠를 놓칩니다.

## 기능 설명
`memsearch`를 통한 벡터 기반 시맨틱 검색:
- OpenClaw 마크다운 메모리 파일을 Milvus 벡터 데이터베이스로 변환
- 시맨틱 쿼리: "어떤 캐싱 솔루션을 선택했지?" → 정확한 단어와 관계없이 관련 내용 찾기
- SHA-256 해싱으로 변경되지 않은 파일 재임베딩 방지 (API 비용 절감)
- 파일 워처로 인덱스 자동 최신화
- OpenAI, Google, Voyage, Ollama, 또는 완전 로컬 임베딩 지원

**설계 원칙:** "마크다운이 진실의 원천. 벡터 인덱스는 파생 캐시 — 언제든 재빌드 가능."

## 필요한 스킬
- `memsearch`
- Python 3.10+
- pip 또는 uv
