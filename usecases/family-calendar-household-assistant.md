# 가족 캘린더 및 가정 어시스턴트

## 문제점
- 업무, 개인, 가족, 학교, 과외 활동 캘린더 파편화
- 산발적인 문자 메시지로 가정 내 협업
- 문자에 답하지 않은 예약 확인

## 기능 설명
- 모든 가족 캘린더 통합 모닝 브리핑 (Google, 공유, 학교 PDF)
- 주변 메시지 모니터링: 15분마다 iMessage 확인, 예약 패턴 감지, 30분 이동 버퍼 포함 캘린더 이벤트 생성
- 식료품 재고: 사진 입력, 냉장고/식품 저장실 추적, 어디서나 조회 가능
- 장보기 조율: 재료 중복 제거, 재고 부족 추적, 쇼핑 목록 생성

## AGENTS.md 설정 예시

```
## 메시지 모니터링 (HEARTBEAT.md)
매 15분:
1. 새 iMessage 확인
2. 예약 유사 패턴 감지
3. 감지 시: 캘린더 이벤트 생성, 앞뒤로 30분 이동 버퍼 추가
4. 가족 Telegram에 확인 전송
```

## 필요한 스킬
- Google Calendar OAuth
- `imessage` 스킬 (macOS 전용)
- Telegram/Slack 가족 채팅
- 파일 시스템
- 카메라/OCR
