# 팟캐스트 제작 파이프라인

## 문제점
솔로 팟캐스터는 녹음보다 제작에 더 많은 시간을 씁니다. 리서치, 쇼노트, 소셜 미디어 홍보가 번거로워 종종 생략됩니다.

## 기능 설명

**녹음 전:**
- 게스트 배경 리서치, 토킹 포인트, 에피소드 개요 및 콜드 오픈 훅
- 인터뷰 질문 5–7개, 여분 질문 2–3개

**녹음 후:**
- 타임스탬프 포함 쇼노트 및 링크
- SEO 최적화 에피소드 설명
- 소셜 미디어 킷: X/Twitter 트윗 3개, LinkedIn 게시물 1개, Instagram 캡션 1개
- 타임스탬프 포함 하이라이트 목록
- 선택: 경쟁 팟캐스트 RSS 모니터링

## 녹음 전 프롬프트

```
[주제]에 대한 팟캐스트 에피소드를 녹음할 예정입니다. 게스트는 [이름]입니다.

다음을 수행해 주세요:
1. 게스트 조사 — 배경, 최근 작업, 핫테이크, 논란이 될 만한 내용
2. 주제 조사 — 주요 트렌드, 최신 뉴스, 일반적인 오해
3. 에피소드 개요 생성:
   - 콜드 오픈 훅 (1–2문장)
   - 인트로 스크립트 (30초)
   - 인터뷰 질문 5–7개
   - 여분 질문 2–3개
   - 콜투액션 포함 클로징

~/podcast/episodes/[에피소드-번호]/prep/에 저장하세요.
```

## 필요한 스킬
- 웹 검색/리서치
- 파일 시스템
- Slack/Discord/Telegram
- `sessions_spawn` (선택)
- RSS 피드 스킬 (선택)
