# X 계정 분석

## 문제점
기존 X 분석 도구는 통계 수치에 집중하고 정성적 패턴은 다루지 않습니다. 구독료도 월 $10–$50 수준입니다.

## 기능 설명
OpenClaw와 Bird 스킬을 활용한 X 계정 정성 분석:
- 내 게시물이 바이럴되는 패턴은?
- 어떤 주제가 가장 많은 인게이지먼트를 얻나?
- 왜 어떤 게시물은 1,000+ 좋아요를 받고 어떤 건 5개도 안 받을까?

## 설정 방법
1. Bird 스킬이 동작하는지 확인
2. 보안·격리를 위해 ClawdBot 전용 X 계정 생성 권장
3. x.com에 Chrome/Brave로 로그인 후 쿠키 정보(`auth-token`, `ct0`) 제공
4. 최근 N개 트윗을 가져와 질문

## 필요한 스킬
- Bird (`clawhub install bird` — 기본 번들 제공)
